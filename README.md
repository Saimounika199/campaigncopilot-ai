# CampaignCopilot AI

AI assistant for marketing operations that transforms vague campaign briefs into launch-ready execution plans.

## Problem
Campaign briefs are often incomplete, ambiguous, and cause delays.

## Solution
CampaignCopilot analyzes briefs, asks clarifying questions, creates execution plans, and performs QA checks.

## Features
- Brief Health Score
- Clarifying Questions
- Execution Plan Generator

## Tech Stack
- Python
- Streamlit
- Azure OpenAI
- GitHub

## Business Impact
Reduces campaign planning time from days to minutes.
